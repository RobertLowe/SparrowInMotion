# README

Demo to use Sparrow Framework - The Open Source Game Engine for iOS - in RubyMotion.

## Install

Clone the project:

```
git clone git://github.com/siuying/SparrowInMotion.git
```

Checkout submodules:

```
git submodule update --init
```

Run it:

```
rake
```
